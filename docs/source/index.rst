Welcome to InfoDESReg documentation!
===================================

InfoDESReg is an open-source library focusing on the implementation of state-of-the-art techniques for dynamic classifier and ensemble selection with late fusion setting. This project is under active development. Contributions are welcomed through its GitHub page: https://github.com/adv-panda/infodesreg
